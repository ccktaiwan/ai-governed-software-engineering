# Governance Adoption Declaration

This repository adopts upstream governance specifications
to regulate AI-assisted engineering activities.

## Upstream Governance

- Repository: ai-governed-software-engineering
- Scope: Engineering lifecycle, AI participation, authority and escalation

## Authority Statement

Upstream governance definitions:
- override local engineering preferences
- must not be reinterpreted as runtime behavior
- apply to all design, review, and implementation activities

## Responsibility Boundary

This repository remains responsible for:
- runtime behavior
- execution correctness
- safety enforcement

Governance repositories define methodology, not execution.

End of Declaration
