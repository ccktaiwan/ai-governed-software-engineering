# Releases

## v0.1

This release establishes the initial version of the
AI-Governed Software Engineering playbook.

Scope:
- Engineering methodology documentation
- Five-phase software development model
- Governed AI participation
- Prospera OS as a live case study

This release does not include:
- Software implementations
- Tooling or automation
- Prescriptive frameworks

v0.1 serves as a foundation for future iteration.

## v0.2

This release formalizes governance structures for AI participation in
software engineering.

Scope:
- AI participation boundary specification
- Authority level allocation between human and AI roles
- Governance enforcement and escalation rules
- Governance index and cross-reference map

This release establishes auditable, bounded, and enforceable governance
without introducing operational procedures, tooling, or implementation guidance.

v0.2 is additive and fully compatible with v0.1.
